# Visir 바이럴 마케팅 전략

*AI-Powered Unified Productivity Platform*

---

## 1. Executive Summary

Visir는 이메일, 메신저, 캘린더, 태스크를 하나의 인터페이스로 통합한 AI 기반 생산성 플랫폼입니다. 바이럴 마케팅의 핵심은 **정보 과부하에 지친 지식 근로자의 공감**을 자극하고, 제품의 핵심 가치인 **Input → Action 전환의 마찰 제거**를 직관적으로 보여주는 것입니다.

## 2. 핵심 메시지 프레임워크

### A. 핵심 가치 제안 (Value Proposition)

| Pain Point | Visir Solution |
|------------|----------------|
| 탭 지옥 (7개 이상의 생산성 앱) | 하나의 창에서 모든 커뮤니케이션과 업무 관리 |
| 메일 읽고 → 캘린더 열고 → 일정 입력 | 드래그 앤 드롭으로 즉시 일정화 |
| "내가 뭘 했더라?" 기억 부담 | AI가 모든 맥락을 기억하고 검색 |

### B. 바이럴 슬로건 후보

| 슬로건 | 포커스 |
|--------|--------|
| "All your inboxes in one. Every message becomes action." | 통합성 강조 |

## 3. 채널별 바이럴 전략

### A. Twitter/X — 공감형 바이럴

**전략:** 테크 종사자들의 "공감 밈"을 활용한 자연스러운 확산

| 포맷 | 설명 |
|------|------|
| Before/After 스레드 | 기존 워크플로우 vs Visir 워크플로우 비교 |
| 공감 트윗 | 탭 7개 상황에 공감대 형성 → 제품 소개로 연결 |
| 데모 GIF | 드래그앤드롭으로 메일→일정 변환하는 3초 클립 |

**KPI:** 리트윗 500+, 스레드 노출 10만+ 목표

### B. LinkedIn — 전문가 신뢰 구축

**전략:** 생산성 전문가, PM, 스타트업 창업자 타겟의 thought leadership

| 포맷 | 설명 |
|------|------|
| 인사이트 포스트 | 생산성 앱 역설에 대한 깊은 분석 |
| 창업자 스토리 | Visir를 만들게 된 개인적 pain point 공유 |
| 사용 사례 | 얼리어답터의 실제 워크플로우 개선 사례 |

**KPI:** engagement rate 5%+, 팔로워 5,000+ 목표

### C. Reddit/커뮤니티 — 얼리어답터 확보

**전략:** ProductHunt, Hacker News, r/productivity 등에서 진정성 있는 론칭

| 플랫폼 | 접근 방식 |
|--------|----------|
| ProductHunt | 론칭 데이 캠페인, 메이커 소개, 커뮤니티 Q&A |
| Hacker News | "Show HN" 포스트로 기술적 차별점 어필 |
| Reddit | r/productivity, r/SaaS에 유용한 팁과 함께 자연스럽게 노출 |

**KPI:** PH Top 5, HN 프론트페이지 달성 목표

### D. TikTok/YouTube — 시각적 데모

**전략:** 짧고 임팩트 있는 제품 데모로 "wait, what?" 반응 유도

| 플랫폼 | 포맷 |
|--------|------|
| TikTok | 15초 Before/After 밈 포맷 |
| YouTube Shorts | 핵심 기능별 60초 튜토리얼 시리즈 |
| YouTube 롱폼 | 전체 워크플로우 데모, 비교 리뷰 (인플루언서 협업) |

**KPI:** TikTok 10만 뷰, YouTube 구독자 1,000+ 목표

## 5. 바이럴 훅 (Viral Hooks)

각 채널에서 공유를 유도하는 핵심 심리적 트리거:

| 트리거 | 설명 |
|--------|------|
| **공감 (Relatability)** | "나만 이런 줄 알았는데..." 반응 유도 |
| **놀라움 (Surprise)** | "이게 된다고?" 드래그앤드롭 → 일정화 |
| **실용성 (Utility)** | "이거 진짜 쓸모있다" 실제 워크플로우 개선 |
| **소속감 (Identity)** | "생산성 덕후들의 새로운 무기" 커뮤니티 형성 |
