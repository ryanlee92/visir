# 플랫폼 아이콘 이미지

이 폴더에 다음 플랫폼의 공식 아이콘 이미지를 넣어주세요:

## 필요한 파일

1. **macos.svg** - macOS 공식 로고 (SVG 형식 권장)
2. **windows.svg** - Windows 공식 로고 (SVG 형식 권장)
3. **ios.svg** - iOS/Apple 공식 로고 (SVG 형식 권장)
4. **android.svg** - Android 공식 로고 (SVG 형식 권장)

## 다운로드 링크

### Apple (macOS/iOS)
- Apple 공식 디자인 리소스: https://developer.apple.com/design/resources/
- Apple 로고 사용 가이드라인: https://www.apple.com/legal/intellectual-property/guidelinesfor3rdparties.html

### Microsoft (Windows)
- Microsoft 브랜드 리소스: https://www.microsoft.com/en-us/legal/intellectualproperty/trademarks/usage/general
- Windows 로고: https://www.microsoft.com/en-us/legal/intellectualproperty/trademarks

### Google (Android)
- Android 브랜드 가이드라인: https://developer.android.com/distribute/marketing-tools/brand-guidelines
- Android 로고: https://developer.android.com/distribute/marketing-tools/brand-guidelines#android-brand

## 파일 형식

- **SVG** 형식을 권장합니다 (확장성과 선명도)
- PNG 형식도 가능하지만, 파일 확장자를 `.png`로 변경하고 코드에서도 `.png`로 수정해야 합니다.

## 사용 방법

1. 위 링크에서 공식 로고를 다운로드하세요
2. 파일 이름을 위에 명시된 대로 정확히 맞춰주세요
3. 이 폴더에 저장하세요
4. `DownloadPage.tsx` 파일에서 import 문의 주석을 해제하세요

## 주의사항

- 각 플랫폼의 브랜드 가이드라인을 준수해야 합니다
- 로고 사용 시 각 플랫폼의 사용 약관을 확인하세요
- 상업적 사용이 허용되는지 확인하세요
