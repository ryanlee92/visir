### 왜 Visir인가요?

- Gmail, Slack, Outlook을 단일 타임라인에서 관리

## 상세 버전 (Extended)

**여러 앱을 오가며 시간을 낭비하지 마세요. 집중력을 되찾으세요.**

Visir는 당신의 AI 전담 비서입니다. 이메일, 캘린더, 팀 채팅을 하나의 지능형 타임라인으로 통합하여 모든 업무를 한 곳에서 관리할 수 있게 해줍니다. 더 이상 앱 사이를 전환할 필요가 없습니다.

### 문제 해결

현대의 업무 환경은 앱으로 가득 차 있습니다. 이메일 앱, 캘린더 앱, 채팅 앱, 작업 관리 앱... 각각을 확인하고 전환하는 데 시간과 정신적 에너지가 소모됩니다. Visir는 이 모든 것을 하나로 통합하여 실제 작업에 집중할 수 있게 해줍니다.

### 통합 인박스

Visir의 통합 인박스는 Gmail, Slack, Outlook을 단일 타임라인으로 통합합니다. 더 이상 여러 앱을 열어가며 확인할 필요가 없습니다. 모든 메시지가 시간순으로 정렬되어 표시되며, 프로젝트별로 자동으로 그룹화됩니다. 이메일과 채팅을 하나의 흐름으로 통합하여 컨텍스트 손실 없이 작업할 수 있습니다.

**주요 기능:**
- 모든 이메일과 채팅 메시지를 하나의 타임라인에서 확인
- 앱을 전환하지 않고 직접 답장 가능
- 프로젝트별 자동 분류 및 그룹화
- 검색 기능으로 빠른 메시지 찾기
- 중요도 기반 자동 정렬

### AI 전담 비서

Visir의 AI는 단순한 도우미가 아닙니다. 당신의 전담 비서처럼 작동합니다. 자연어로 요청하면 AI가 모든 것을 처리합니다.

**AI의 능력:**
- **하루 요약**: 아침에 하루 일정과 중요한 업데이트를 요약해 제공
- **작업 정리**: 인박스를 분석하여 작업을 프로젝트별로 정리하고 우선순위 설정
- **회의 준비**: 다가오는 회의의 핵심 컨텍스트를 요약하여 준비 시간 단축
- **회의 재조정**: 자연어로 요청하면 AI가 회의 일정을 재조정하고 관련자에게 알림
- **스마트 트리아지**: 긴급한 신호와 낮은 우선순위의 소음을 자동으로 구분하여 중요한 것에 집중
- **컨텍스트 인식 응답**: 스레드의 맥락을 이해하고 적절한 응답 초안 작성

**홈 탭의 지능형 브리핑:**
홈 탭은 단순한 목록이 아닙니다. AI가 인박스를 분석하여 프로젝트와 긴급도에 따라 모든 항목을 분류하는 지능형 브리핑입니다. 하루를 시작하기 전에 무엇이 중요한지, 무엇을 먼저 처리해야 하는지 명확하게 알 수 있습니다.

### 완전한 기능을 갖춘 수직 탭

대부분의 생산성 앱들은 얕은 통합만 제공합니다. Visir는 다릅니다. 각 탭(Mail, Chat, Calendar, Tasks)은 독립적인 앱처럼 완전한 기능을 제공하면서도 완벽하게 통합되어 있습니다.

**Mail 탭:**
- 완전한 이메일 클라이언트 기능
- 읽기, 작성, 답장, 전달
- 첨부파일 관리

- 오프라인 접근

**Chat 탭:**
- Slack, Microsoft Teams 등 팀 채팅 통합

- 파일 공유 및 협업
- 스레드 관리
- 알림 설정

**Calendar 탭:**
- Google Calendar, Outlook Calendar 통합

- 회의 초대 및 응답
- 반복 일정 관리
- 타임존 지원

**Tasks 탭:**
- 작업 생성 및 관리
- 프로젝트별 그룹화
- 우선순위 설정
- 마감일 추적
- 체크리스트 및 서브태스크

### 타임 블로킹

Visir는 타임 블로킹을 위해 특별히 설계되었습니다. 전통적인 캘린더 앱들은 타임 블로킹에 적합하지 않습니다. Visir는 드래그 앤 드롭으로 간단하게 작업을 캘린더에 블록할 수 있게 해줍니다.

**작동 방식:**
1. 이메일이나 작업을 캘린더로 드래그
2. Visir가 자동으로 캘린더 이벤트 생성
3. 작업과 캘린더 이벤트가 자동으로 연결
4. 변경 사항이 양쪽에 자동 반영

**고급 기능:**
- 스마트 제안: 과거 패턴, 에너지 수준, 관련 회의를 기반으로 작업 블록 제안
- 자동 블록 생성: 반복 작업에 대한 자동 블록 생성
- 실시간 적응: 긴급한 일이 생기면 블록을 드래그하여 재배치 가능
- 컨텍스트 표시: 작업과 관련된 이메일, 채팅 메시지, 다른 작업을 함께 표시
