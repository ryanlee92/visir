## 작업 관리

### 작업 생성

#### 기본 작업 생성
```
사용자: "내일 오전 9시에 회의 준비 작업 만들어줘"
AI: 작업을 생성합니다.
→ createTask 함수 호출
```

#### 상세 정보 포함 작업 생성
```
사용자: "프로젝트 문서 작성 작업을 만들어줘. 설명은 'API 문서 작성 및 리뷰'로 하고, 내일 오전 9시부터 오후 5시까지로 설정해줘"
AI: 상세 정보를 포함한 작업을 생성합니다.
→ createTask 함수 호출 (title, description, startAt, endAt 포함)
```

#### 하루 종일 작업 생성
```
사용자: "오늘 하루 종일 '휴가 신청서 작성' 작업 만들어줘"
AI: 하루 종일 작업을 생성합니다.
→ createTask 함수 호출 (isAllDay: true)
```

#### 프로젝트 지정 작업 생성
```
사용자: "'프로젝트 A' 프로젝트에 '디자인 리뷰' 작업 추가해줘"
AI: 지정한 프로젝트에 작업을 추가합니다.
→ createTask 함수 호출 (projectId 포함)
```

### 작업 수정

#### 작업 제목 수정
```
사용자: "방금 만든 작업 제목을 '긴급: 회의 준비'로 바꿔줘"
AI: 작업 제목을 수정합니다.
→ updateTask 함수 호출 (태그된 작업 사용)
```

#### 작업 날짜 변경
```
사용자: "이 작업을 내일로 옮겨줘"
AI: 작업 날짜를 변경합니다.
→ updateTask 함수 호출 (startAt, endAt 업데이트)
```

#### 작업 상태 변경
```
사용자: "이 작업 완료 처리해줘"
AI: 작업을 완료 상태로 변경합니다.
→ toggleTaskStatus 함수 호출
```

### 작업 삭제

```
사용자: "이 작업 삭제해줘"
AI: 작업을 삭제할까요? (확인 요청)
→ 사용자 확인 후 deleteTask 함수 호출
```

```
→ updateEvent 함수 호출 (태그된 일정의 eventId 사용)
```

### 태그된 이메일 답장

```
사용자: (이메일을 태그한 후) "이 이메일에 답장해줘"
AI: 태그된 이메일에 답장합니다.
→ replyMail 함수 호출 (태그된 이메일의 threadId 사용)
```

## 팁과 요령

### 1. 구체적인 요청하기

**좋은 예:**
- "내일 오전 9시부터 오후 5시까지 '프로젝트 문서 작성' 작업 만들어줘"
- "우리카드에서 온 메일 찾아서 답장해줘. 내용은 '확인했습니다.'로 해줘"

**나쁜 예:**
- "작업 만들어줘" (제목, 날짜 등 정보 부족)
- "메일 답장해줘" (어떤 메일인지 불명확)

### 2. 날짜 표현

AI는 다양한 날짜 표현을 이해합니다:
- "오늘", "내일", "다음 주 월요일"
- "2024년 1월 1일"
- "1주일 후", "3일 후"

### 3. 여러 작업 요청

한 번에 여러 작업을 요청할 수 있습니다:
- "오늘 '이메일 확인', '문서 작성', '회의 준비' 작업 3개 만들어줘"
- "이 작업들 모두 완료 처리해줘"

### 4. 검색 후 작업

검색과 작업을 한 번에 요청할 수 있습니다:
- "우리카드에서 온 메일 찾아서 답장해줘"
- "'회의 준비' 작업 찾아서 완료 처리해줘"

### 5. 확인이 필요한 작업

다음 작업은 확인 후 실행됩니다:
- 이메일 전송/답장/전달
- 작업/일정/이메일 삭제
- 작업/일정 수정
- 작업/일정 생성

확인 화면에서 내용을 검토한 후 실행하거나 취소할 수 있습니다.

### 6. 프로젝트 컨텍스트 활용

프로젝트를 선택한 상태에서 질문하면 해당 프로젝트의 작업 정보를 활용합니다:
- "이 프로젝트에서 어떤 작업을 하고 있어?"
- "프로젝트 요약해줘"

### 7. 태그된 항목 활용

작업, 일정, 이메일을 태그한 후 요청하면 해당 항목을 자동으로 사용합니다:
- (작업 태그 후) "이 작업 완료 처리해줘"
- (일정 태그 후) "이 일정 삭제해줘"

## 자주 묻는 질문 (FAQ)

### Q: AI가 내 요청을 제대로 이해하지 못했어요.

**A:** 더 구체적으로 요청해보세요:
- 날짜와 시간을 명시하세요
- 작업 제목과 설명을 포함하세요
- 검색할 키워드를 명확히 하세요

### Q: 여러 작업을 한 번에 만들 수 있나요?

**A:** 네, 가능합니다:
- "오늘 '작업1', '작업2', '작업3' 만들어줘"
- "오늘부터 매일 1주일 동안 '운동' 작업 만들어줘"

### Q: 검색 결과를 어떻게 사용하나요?

**A:** 검색 결과는 자동으로 컨텍스트에 추가됩니다:
- 검색 후 바로 작업을 수행할 수 있습니다
- "우리카드에서 온 메일 찾아서 답장해줘"와 같이 한 번에 요청 가능

### Q: 작업을 삭제했는데 복구할 수 있나요?

**A:** 현재는 삭제된 작업을 복구하는 기능이 없습니다. 삭제 전에 확인 화면에서 취소할 수 있습니다.

### Q: 반복 작업을 만들 수 있나요?

**A:** 네, 가능합니다:
- "오늘부터 매일 1주일 동안 '운동' 작업 만들어줘"
- "매주 월요일 오전 10시에 '팀 미팅' 일정 만들어줘"

### Q: 프로젝트 정보를 어떻게 활용하나요?

**A:** 프로젝트를 선택한 상태에서 질문하면 해당 프로젝트의 작업 정보를 분석합니다:
- "이 프로젝트에서 어떤 작업을 하고 있어?"
- "프로젝트 요약해줘"
